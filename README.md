
# 交易信号后台（Web版）

这是一个 Flask 网页后台，适合你和助理在浏览器里使用。

## 功能
- 新增吞没信号记录
- 按助理 / 品种 / 结构位 / 方向筛选
- 统计总信号、命中率、平均 MFE、平均 RR
- 图表展示：
  - 各结构位信号数量
  - 各助理命中率
- 导出 CSV
- 删除记录

## 运行方式

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 启动程序
```bash
python app.py
```

3. 浏览器打开
```text
http://127.0.0.1:5000
```

## 数据库
会自动生成：
- signals_web.db

## 建议用法
- 你自己做管理员
- 助理只负责录入和补数据
- 每周导出一次 CSV 做进一步分析
